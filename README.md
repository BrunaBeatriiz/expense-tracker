## 🇧🇷 Sobre o Projeto

Aplicação de controle de gastos pessoais desenvolvida com React, com foco em organização financeira e visualização de dados.  
Permite ao usuário registrar despesas, filtrar informações e acompanhar seus gastos ao longo do tempo.

---

## 🇺🇸 About the Project

A personal expense tracker built with React, focused on financial organization and data visualization.  
Users can add expenses, filter data, and track their spending over time.

## 💻 Como executar | How to run

```bash
# Clone o repositório
git clone https://github.com/BrunaBeatriiz/expense-tracker.git

# Acesse a pasta
cd expense-tracker

# Instale as dependências
npm install

# Execute o projeto
npm run dev

```
